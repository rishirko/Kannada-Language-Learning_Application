# Kannada-Language-Learning-Application
This app displays lists of vocabulary words for the user to learn the Kannada language.
Tapping on any word will play the audio of the pronunciation of the word.

Pre-requisites
--------------

- Android SDK v31
- Android Build Tools v31.0.2
- Android Support Repository v31.3.0

Getting Started
---------------

This sample uses the Gradle build system. To build this project, use the
"gradlew build" command or use "Import Project" in Android Studio.
